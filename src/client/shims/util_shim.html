inspect gives you a way better representation of an object than JSON.stringify. You can provide a higher maxDepth to go deeper into an object. Strings returned look like this:

{
  left: "yes",
  right: {
    left: "yes",
    right: 'no'
  }
}

or, special-cased output for Error objects:

[error: You can't do that!]
